# Feature Matrix: Free vs Premium

Clear breakdown of features available in each tier with rationale.

## Philosophy

**Free Tier Goal**: Provide core value (basic speed tracking) to attract users and demonstrate quality. Enable users to experience the app's accuracy and interface before committing to a subscription.

**Premium Tier Goal**: Unlock advanced features that provide deeper insights, better usability, and data persistence. Premium should feel like a natural upgrade, not a paywall blocking basic functionality.

**Conversion Strategy**: Free users get the "what" (current speed), Premium users get the "why" and "how" (analytics, history, advanced modes).

## Free Tier Features (Detailed)

### What Free Users Get:

1. **Real-Time Speed Tracking**
   - Current speed display
   - Speed unit selection (mph/km/h)
   - GPS accuracy indicator
   - Basic black/white theme only

2. **Speed Limit Alerts**
   - Set maximum speed limit
   - Set minimum speed limit
   - Sound and vibration alerts
   - Mute controls
   - Visual alerts

3. **Limited Trip History**
   - View last 5 trips only
   - Basic trip info (date, duration, distance)
   - No maps or graphs
   - No export capability
   - Trips auto-delete when exceeding 5

4. **Essential Settings**
   - Language selection
   - Speed unit preference
   - Alert preferences
   - Account management
   - Support links

5. **Apple ID Integration**
   - Sign in with Apple
   - Basic profile
   - Preference sync via iCloud

### What Free Users See (Upsell Moments):

- "Upgrade" badge on locked features
- Paywall when tapping HUD mode button
- Paywall when tapping Pedometer tab
- "Unlock unlimited history" banner in History
- "Premium" badges on locked settings
- Upgrade button in Settings prominently placed

## Upgrade Triggers (When Paywall Appears)

1. **From Onboarding**: After feature walkthrough (soft sell, dismissible)
2. **HUD Mode Button**: When free user taps HUD button
3. **Pedometer Tab**: When free user taps Pedometer tab
4. **Trip #6**: When trying to view 6th trip in history
5. **Export Button**: When trying to export trip data
6. **Color Theme**: When accessing theme customization
7. **Weather Widget**: When tapping weather area (if shown)
8. **Settings Upgrade Button**: Direct access from settings
9. **History Banner**: "Unlock unlimited history" call-to-action

## Feature Gating Strategy

### Soft Gates (Dismissible)
- Onboarding paywall: Can skip
- History upgrade banner: Can ignore
- Settings upgrade button: Can ignore

### Hard Gates (Required Premium)
- HUD mode: Immediate paywall, no access without premium
- Pedometer: Immediate paywall
- Trip #6+: Cannot view without upgrade
- Weather: Feature hidden in free tier
- Compass: Feature hidden in free tier
- Color themes: Locked in settings
- Export: Button disabled with premium badge

## Conversion Optimization

### Free Trial Strategy
- Offer 3-day free trial only on yearly plan
- Maximizes trial period value
- Encourages annual commitment
- Reduces refund requests

### Feature Reveal Strategy
1. **Week 1**: User experiences free tier, sees quality
2. **Week 2**: User hits trip limit, sees history value
3. **Week 3**: User wants HUD mode, sees premium features
4. **Month 1**: User is habitual, ready to commit to yearly

### Retention Strategy
- Premium users: Full feature access forever
- Lapsed premium: Can still view old history (read-only)
- Re-subscription: Restore full access immediately

## Premium Feature Justification

| Feature | Why Premium? |
|---------|--------------|
| HUD Mode | Advanced UI mode requiring special implementation |
| Full Dashboard | Complex data processing and display |
| Unlimited History | Storage costs, iCloud sync complexity |
| Trip Maps | MapKit integration, rendering costs |
| Graphs | Analytics processing |
| Pedometer | Separate CoreMotion integration |
| Weather | API costs (if using third-party) |
| Color Themes | Customization = premium perk |
| Export | Power user feature |
| iCloud Sync | Server-side costs (iCloud storage) |

All premium features represent either:
1. **Development complexity** (HUD, maps, graphs)
2. **Ongoing costs** (weather API, iCloud)
3. **Power user needs** (export, unlimited storage)
4. **Premium experience** (customization, advanced modes)

This justifies the subscription model and ensures free tier remains usable.
